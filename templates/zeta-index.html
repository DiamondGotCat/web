What is "Fully" Open-source?

Zeta's biggest features

The Open Source LLM can't really be called open source.

DiamondGotCat, the developer of Zeta, noticed this.

Open source LLM is just a model released under an open source license.

Where is the source? There is no complete source anywhere.

Zeta is "Fully" Open-source LLM.

To solve this problem, Zeta has made everything public,

including not only the models but also the training scripts (which we actually use) and datasets.

How does it compare to other LLMs?

Lightweight and efficient

Accuracy of answers

To be honest, Zeta's answer is not that great.

However, considering that Zeta has parameters of about 0.5B,

and was created by an individual, it's not bad.

Support for GGUF format

All models have been converted by mradermacher and are of high quality and easy to try.

It is also available on Ollama, so please feel free to use it.
